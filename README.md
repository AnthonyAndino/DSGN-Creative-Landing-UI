# DSGN-Creative-Landing-UI

## 🇪🇸 Español

Interfaz moderna de un estudio creativo diseñada con HTML y CSS puro.
Incluye landing page, login y registro con estilo glassmorphism y efectos neón.

---

### Características

* Diseño oscuro moderno (Dark UI)
* Efectos visuales tipo neon
* Glassmorphism en formularios
* Responsive (base adaptable)
* Formularios de Login y Registro
* Barra de búsqueda en landing

---

### Tecnologías utilizadas

* HTML5
* CSS3
* Font Awesome
* Google Fonts (Montserrat & Poppins)

---

### Nota

Este proyecto es parte de mi portafolio como desarrollador en crecimiento.
